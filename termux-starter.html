const VanillaServer = require('./vanillaserver.js');

Step 3: Java Server — The Lion

Install JDK 17

pkg install openjdk-17

Verify: java -version should show 17.x

Why Java on Phone?

  • Concurrency: Threads, locks, synchronized. King commands armies.
  • No runtime deps: JDK ships with com.sun.net.httpserver. No Spring. No Tomcat.
  • Performance: JIT compiler. Outruns interpreted scripts under load.
  • ACID: Connect to PostgreSQL with raw JDBC. He Goat approved.

Compile + Run Java HTTP Server

// Server.java
import com.sun.net.httpserver.HttpServer;
import com.sun.net.httpserver.HttpExchange;
import java.net.InetSocketAddress;
import java.io.OutputStream;
import java.nio.charset.StandardCharsets;

public class Server {
  public static void main(String[] args) throws Exception {
    HttpServer server = HttpServer.create(new InetSocketAddress(8080), 0);
    
    server.createContext("/", ex -> {
      String res = "<h1>Lion Live</h1><p>Java 21. Zero deps. Fears no load.</p>";
      send(ex, 200, res);
    });
    
    server.createContext("/api/health", ex -> {
      String json = "{\"status\":\"ok\",\"engine\":\"jdk\",\"deps\":0}";
      ex.getResponseHeaders().set("Content-Type", "application/json");
      send(ex, 200, json);
    });
    
    server.setExecutor(null);
    server.start();
    System.out.println("Lion running at http://localhost:8080");
  }
  
  static void send(HttpExchange ex, int code, String res) throws Exception {
    byte[] bytes = res.getBytes(StandardCharsets.UTF_8);
    ex.sendResponseHeaders(code, bytes.length);
    try (OutputStream os = ex.getResponseBody()) {
      os.write(bytes);
    }
  }
}
javac Server.java
java Server

Open Chrome: http://127.0.0.1:8080. That's raw JDK. No framework. Compiles in 200ms. Runs on 300MB RAM.

Java Story: Spring Boot = 50MB minimum, 8s cold start. This = 50KB source, 200ms start. Termux can't run Spring. It can run the Lion. Hard Knocks doesn't need annotations. It needs synchronized.

  1. Static sites: Upload .html files to Netlify. Drag-drop. Done.
  2. Node API: Deploy to Render/Railway. Set start command: node server.js
  3. Java API: Compile to JAR: jar cfe app.jar Server *.class. Deploy to Fly.io or any VPS.
